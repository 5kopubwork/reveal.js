Ecosystem Updates

## New Core Concept in SME Central A new Core Concept has been added to the SME Central for **Documentation as Code**. With the push of DXC **going digital**, one concept that will greatly help us get there is the Documentation as Code. Visit the new [Documentation as Code portal](https://github.dxc.com/pages/AET/EDGE-Central/Pages/DocAsCode_Home.html) to learn more or watch our Documentation as Code [quick videos](https://web.microsoftstream.com/channel/17b10bae-664c-4866-84f9-7f97046a1a4e).

TIPs - New Menu Option

  • TIPs has added as link to EDGE Process Framework on it’s top bar menu. This provides easy access to the a wide range of process information, assets and collaterals from EDGE.

TIPs - Domain Navigation Change

  • For a more accurate and efficient distribution and grouping of TIPs assets, we have renamed the Process Framework tab to Knowledge & Communities. The new tab will contain the process assets, community assets and knowledge-base assets for each domain.

TIPs Improvements and Updates

  • Continuous monitoring of domains and portals to make sure that TIPs contents are always accurate.
  • Each domain is assigned to a SME to monitor for updates and to work key stakeholders.

Next Steps

  • Further publicize content
  • Solicit feedback – crowd source
  • Build new relationships for new content
