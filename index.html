What is Documentation as code?

## What's in it for DXC? - Increase margins, reduce operational costs, drive higher collaboration and reuse to reduce TTM - Increase in cross-account collaboration potential - Convergence on a single platform for revision control and collaboration - Enables ability to access any content and have a conversation without artificial barriers - Leverage examples without reinventing the wheel


Examples?

## GitHub Pages Template - GitHub provides static web hosting, generating web content from text-based data stored in GitHub. - View sample site based on Minimal Mistakes theme [here](https://github.dxc.com/pages/AET/DocAsCodeSamples/). - View source for site in root of repo (all folders excluding those prefixed with *ex*). - Site is a replica of the [OurSite template](https://github.dxc.com/AET/OurSite), which can be leveraged as needed and is currently used by many teams as the basis of their own sites.
## js sequence diagrams - View an example of a UML diagram generated from text-based syntax and then rendered through GitHub pages using javascript included with the sample. - View generated documentation as code **diagram** [here](https://github.dxc.com/pages/AET/DocAsCodeSamples/ex_diagrams/index.html). - Update this [file](https://github.dxc.com/lsheridan2/DocAsCodeSamples/blob/master/ex_diagrams/index.html) to change the diagram. - Based on js sequence diagrams - [more info](https://bramp.github.io/js-sequence-diagrams/). - Click [here](https://github.com/bramp/js-sequence-diagrams/blob/master/src/grammar.jison) for more info on syntax.
## reveal js presentations - View an example of a presentation rendered from text-based content stored in GitHub and then rendered through GitHub pages using javascript included with the sample. - View generated documentation as code **presentation** [here](https://github.dxc.com/pages/AET/DocAsCodeSamples/ex_presentations/presentations/MyPresentation.html). - Update this [file](https://github.dxc.com/lsheridan2/DocAsCodeSamples/blob/master/ex_presentations/presentations/MyPresentation.html) to change the diagram. - Based on reveal js presentations - [more info](https://github.com/hakimel/reveal.js). - Click [here](https://github.com/hakimel/reveal.js#instructions) for more info on syntax.
